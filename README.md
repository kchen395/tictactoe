# tictactoe

## Instructions

Set up postgreSQL and create a database named "tictactoe"

In root folder:

Install dependecies:
```sh
npm install
```

Compile bundle: 
```sh
webpack
```

Run application:
```sh
npm start
```

Access the application at (http://localhost:3000).

Other players on the same local network can play by using the IP address of the machine running the application.
